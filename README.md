# Arch Linux ARM for Pixel 3a (Sargo)

## What is this?
This project contains the necessary scripts, configuration, and packages to build and run a custom Arch Linux ARM (aarch64) environment on a Google Pixel 3a (codename "sargo").

## Why did I do this?
I built this project just for fun! I wanted to try using this phone as my main laptop to test the waters before fully switching my main laptop to an ARM-based one. 

## Building and Flashing
If you want to build the rootfs and boot images yourself and flash them to your device, please check out the detailed guide in the `scripts/create-imgs` directory:
**[Build & Flash Instructions](scripts/create-imgs/README.md)**

## Custom Packages
The rootfs generated by this project relies on custom packages hosted on my personal web repository to make things work out of the box. 

If you prefer not to trust my pre-compiled binaries, you don't have to! You can build them entirely yourself. The `pkgbuilds` folder contains all the PKGBUILD files required to compile everything from source on your own.
